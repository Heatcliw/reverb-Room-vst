# Reverb-Room-vst

Физически-ориентированный аудиоэффект реверберации, моделирующий распространение звука в помещении на основе геометрии комнаты, материалов поверхностей и пространственного расположения источника звука.

## Описание

Большинство цифровых ревербераторов используют набор задержек, фильтров и обратных связей для создания эффекта пространства. Данный проект ставит целью приблизить поведение эффекта к реальной акустике помещения. 

**Плагин**: Visual Box Reverb DSP v1.0.0  
**Формат**: VST 3.6.14  
**Платформа**: Windows x86-64  
**Разработчик**: DSP Engineer

Пользователь задаёт:
- размеры комнаты (ширина, длина, высота)
- материалы стен, пола и потолка
- положение источника звука
- направление источника звука
- положение слушателя

На основе этих данных движок вычисляет ранние отражения, параметры затухания и характеристики поздней реверберации.

## Возможности

### Геометрия помещения

Поддерживается настройка:
- ширины комнаты
- длины комнаты
- высоты комнаты

Размеры помещения влияют на время прихода отражений и общую акустику пространства.

### Материалы поверхностей

Для каждой поверхности задаются акустические свойства:
- коэффициент поглощения
- коэффициент рассеивания
- шероховатость поверхности

Поддерживаются типовые материалы:
- бетон
- кирпич
- дерево
- штукатурка
- ковровое покрытие
- акустические панели

### Пространственное расположение

Источник звука задаётся координатами: X, Y, Z  
Также может быть задано направление излучения: Yaw, Pitch  
Слушатель также имеет собственное положение в пространстве.

### Ранние отражения

Для расчёта первых отражений используется метод зеркальных источников (Image Source Method). Это позволяет учитывать:
- расстояние до поверхностей
- задержку отражений
- потери энергии при отражении
- изменение характера звучания в зависимости от положения источника

### Поздняя реверберация

Хвост реверберации формируется на основе вычисленных акустических параметров помещения с использованием:
- FDN (Feedback Delay Network)
- Schroeder Reverb
- гибридных методов моделирования

### RT60

Время затухания помещения вычисляется автоматически на основе объёма комнаты и коэффициентов поглощения поверхностей.

## Установка

### Требования

- **ОС**: Windows 10 или выше
- **Архитектура**: x86-64
- **DAW (Digital Audio Workstation)**: любая DAW с поддержкой VST3
  - Reaper
  - FL Studio (версия 20.76+)
  - Cubase (версия 11+)
  - Studio One
  - BitWig Studio
  - и другие

### Инструкция по установке

1. **Скачайте файл плагина**
   - Файл: `room_reverb_vst.dll`

2. **Переименуйте файл в VST3 формат**
   ```
   room_reverb_vst.dll  →  room_reverb_vst.vst3
   ```
   
   **Важно**: VST3 плагины на Windows должны иметь расширение `.vst3`

3. **Поместите плагин в папку VST3**
   
   Стандартная папка для VST3 плагинов на Windows:
   ```
   C:\Program Files\Common Files\VST3\
   ```
   
   Или используйте папку вашей DAW:
   - **Reaper**: `C:\Program Files\REAPER\Plugins\VST3\`
   - **FL Studio**: `C:\Program Files\Image-Line\FL Studio 21\Plugins\VST3\`
   - **Cubase**: `C:\Program Files\Steinberg\Cubase 12\VST3\`
   - **Studio One**: `C:\Program Files\PreSonus\Studio One 6\VST3 Plug-Ins\`

4. **Проверьте установку**
   - Запустите вашу DAW
   - Перезагрузите список плагинов (обычно это делается через меню)
   - Найдите плагин "Visual Box Reverb DSP" в списке эффектов
   - Плагин должен отображаться в категории "Reverb"

5. **Используйте плагин**
   - Добавьте плагин на трек или мастер-канал
   - Настройте параметры комнаты, материалы и положение источника
   - Отрегулируйте глубину эффекта (Reverb Time)

### Решение проблем

**Плагин не отображается в DAW**
- Убедитесь, что файл имеет расширение `.vst3` (не `.dll`)
- Проверьте, что файл находится в папке VST3
- Перезагрузите DAW
- В некоторых DAW нужно явно добавить путь к папке VST3 в настройках

**Ошибка при загрузке плагина**
- Убедитесь, что вы используете Windows 10 или выше
- Убедитесь, что установлены необходимые компоненты Windows
- Попробуйте переместить плагин в другую папку VST3
- Проверьте, что ваша DAW поддерживает VST3

**Плагин работает нестабильно**
- Обновите вашу DAW до последней версии
- Проверьте, что у вас достаточно оперативной памяти
- Убедитесь, что ваш процессор поддерживает необходимые инструкции
